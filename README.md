## 🖥️ What Audit-Checklists Does

Audit-Checklists is a set of ready-made security audit checklists for offline use. It helps you work through checks in a clear order. It includes commands, notes, and tool support for common target types like Windows, Linux, Docker, Kubernetes, AWS, Azure, GCP, and containers

This app is made for penetration testers and red teamers, but it can also help anyone who needs a structured security review on a test system
